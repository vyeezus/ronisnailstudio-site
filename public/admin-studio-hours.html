Studio booking hours

These hours control which days and time ranges appear on the public booking calendar. Saving updates the StudioHours tab in your Google Sheet.

One-time setup in Google Apps Script: run setBookingAdminSecret (replace the placeholder string with your own secret, then Run). The secret is stored in Script properties, not in this page. Optional: run seedStudioHoursSheet to pre-fill the sheet from the old defaults.
