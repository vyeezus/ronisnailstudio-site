Studio booking hours

Book a client (admin) · Home

These hours control which days and time ranges appear on the public booking calendar. Saving updates the StudioHours tab in your Google Sheet.

The Save button talks to Google Apps Script, not only this website. Copy the latest google_apps_script.js from your repo into the Apps Script editor, save, then Deploy → Manage deployments → Edit → Version: New version → Deploy. One-time: run setBookingAdminSecret (put your secret in the code, then Run). Optional: seedStudioHoursSheet.
